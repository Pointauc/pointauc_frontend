### Background Transparency

Control how see-through your overlay background is:

- **0% (Solid)**: Completely opaque background
- **50% (Semi-transparent)**: Partially see-through
- **100% (Transparent)**: Completely transparent background

### Table Settings

Fine-tune how the auction table behaves on stream:

- **Autoscroll**: Enable to automatically scroll the table. Disable to keep the table static.
- **Autoscroll Speed**: Adjust how fast the table scrolls when autoscroll is enabled. This setting is only available when **Autoscroll** is turned on.
